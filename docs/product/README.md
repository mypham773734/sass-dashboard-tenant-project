## 🔄 Workflow

For each new feature:

1. **Create folder** under `docs/product/FEATURE_NAME/`
2. **Write 4 documents** following the standard:
   - 00-OVERVIEW.md
   - 01-REQUIREMENTS.md
   - 02-ARCHITECTURE.md
   - 03-IMPLEMENTATION_PLAN.md
3. **Submit for review** (tech lead, architect)
4. **Get approval** before coding
5. **Implement** using the plan
6. **Update docs** with lessons learned

**→ Read [../DOCUMENTATION_STANDARD.md](../DOCUMENTATION_STANDARD.md) for detailed guidelines**
